# 💻 JobHunt Tracker Frontend Web Client

This directory contains a skeleton React application set up with Vite. 

## 🚀 Future Development

The frontend is currently a placeholder for a future web-based dashboard companion to the Chrome Extension.

To launch the dev server:
```bash
npm install
npm run dev
```
Runs a local development server on `http://localhost:5173` (or the next available port).
